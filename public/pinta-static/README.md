Статичная версия лендинга "Пинта".

Файлы:
- `index.html` — главная страница (статичная)
- `css/style.css` — стили
- `js/main.js` — простые анимации и счётчики

Замените плейсхолдеры изображений, загрузив свои файлы в:
`/public/assets/images/pinta/` и обновите пути в `index.html`, если требуется.

Интеграция в проект:
- Можно подключить содержимое `index.html` как отдельную страницу (например, /pinta-static/), либо перенести разметку в шаблон `app/Views/home/index.php`.
- Не забудьте скопировать `css/style.css` в `/public/assets/css/` или обновить ссылку в шаблоне.
